Gateway DNS-over-HTTPS dengan pemblokir iklan & pelacak, optimasi ECS, pengalihan CNAME, dan geo-bypass dinamis di Edge Network.

Menghubungkan...
🚫

Ad & Tracker Blocking

Memblokir iklan dan pelacak menggunakan daftar tepercaya AdGuard, ABPVN, dan 1Hosts. Diperbarui otomatis setiap jam.

🌍

ECS Location Optimization

Mengirimkan lokasi jaringan terdekat secara cerdas (Indonesia & global) agar CDN mengembalikan server tercepat.

🔄

DNS Redirect

Mengarahkan kueri domain tertentu secara instan ke CNAME CDN terbaik di jaringan tepi.

🛡️

Geo-Bypass

Mendeteksi pemblokiran regional (sensor/merah) dan otomatis mengalihkan kueri ke Mullvad DNS tepercaya.

⚡

Query Filtering

Menyaring tipe kueri yang tidak perlu (AAAA, PTR, HTTPS) di jaringan tepi untuk mereduksi beban upstream.

🔐

Private TLD Block

Mencegah kebocoran kueri TLD privat seperti .local, .internal, atau .test ke internet publik.

💻 Konfigurasi Browser (Chrome / Edge / Firefox)
  • 1 Buka halaman Pengaturan (Settings) → Privasi & Keamanan.
  • 2 Aktifkan fitur Gunakan DNS Aman (Use Secure DNS).
  • 3 Pilih opsi Kustom (Custom / Enter URL) dan tempel URL berikut:
    https://...
🍎 Apple (iOS / macOS)
  • 1 Buka halaman ini menggunakan browser Safari.
  • 2 Unduh berkas konfigurasi profil DoH: Unduh Profil (.mobileconfig).
  • 3 Buka Pengaturan Sistem → Profil Terunduh, lalu instal dan aktifkan di Pengaturan VPN & DNS.
📱 Android
  • 1 Unduh aplikasi gratis Intra dari Google Play Store.
  • 2 Masuk ke menu Pengaturan aplikasi Intra, pilih Custom server URL.
  • 3 Masukkan URL DoH Gateway Anda dan aktifkan koneksi:
    https://...
